## 一、设计原则

### 1.1 核心原则

| 原则 | 说明 | 实践要点 |
|------|------|----------|
| **一目了然** | 核心信息3秒内可理解 | 突出关键指标，减少视觉干扰 |
| **层次分明** | 信息有主次之分 | 大→中→小的视觉层级 |
| **行动导向** | 数据驱动决策 | 异常高亮，提供下钻入口 |
| **场景适配** | 符合使用场景 | 大屏/桌面/移动端差异化设计 |

### 1.2 设计流程

```
明确目标 → 定义指标 → 选择图表 → 设计布局 → 交互设计 → 验证优化
```

## 二、看板类型与场景

### 2.1 按使用场景分类

| 类型 | 使用场景 | 更新频率 | 典型用户 |
|------|----------|----------|----------|
| **实时监控看板** | 运维监控、活动监控 | 秒级/分钟级 | 运维、运营 |
| **运营日报看板** | 日常运营跟踪 | 每日 | 运营、产品 |
| **管理决策看板** | 战略决策支持 | 每周/每月 | 管理层 |
| **分析探索看板** | 深度数据分析 | 按需 | 数据分析师 |
| **对外展示看板** | 品牌/成果展示 | 实时/定期 | 公众、客户 |

### 2.2 按业务领域分类

| 业务领域 | 核心指标 | 典型看板 |
|----------|----------|----------|
| 用户增长 | DAU、MAU、新增、留存 | 增长漏斗、留存曲线 |
| 商品运营 | GMV、转化率、客单价 | 销售趋势、商品排行 |
| 内容运营 | PV、UV、互动率、完播率 | 内容分发、用户行为 |
| 客户服务 | 响应时间、解决率、满意度 | 工单监控、服务质量 |
| 技术运维 | 可用性、响应时间、错误率 | 系统监控、告警中心 |

## 三、指标设计

### 3.1 指标选择原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **相关性** | 与业务目标直接相关 | 电商看GMV而非页面浏览量 |
| **可行动** | 数据变化可指导行动 | 转化率下降→优化漏斗 |
| **可衡量** | 有明确的计算方式 | DAU = 当日登录用户数 |
| **可比较** | 支持时间/维度对比 | 环比、同比、分组对比 |

### 3.2 指标层级设计

```
北极星指标 (1个)
    │
    ├── 一级指标 (3-5个)
    │       │
    │       ├── 二级指标 (每个一级下2-5个)
    │       │
    │       └── 三级指标 (细分维度)
    │
    └── 辅助指标 (支撑分析)
```

### 3.3 常用指标组合

#### 用户增长看板
| 层级 | 指标 | 说明 |
|------|------|------|
| 北极星 | MAU | 月活跃用户数 |
| 一级 | DAU、新增用户、留存率、活跃率 | 核心增长指标 |
| 二级 | 各渠道新增、各版本留存、用户时长 | 细分维度 |

#### 电商运营看板
| 层级 | 指标 | 说明 |
|------|------|------|
| 北极星 | GMV | 商品交易总额 |
| 一级 | 订单量、客单价、转化率、复购率 | 核心交易指标 |
| 二级 | 品类GMV、渠道转化、用户分层 | 细分维度 |

#### 内容运营看板
| 层级 | 指标 | 说明 |
|------|------|------|
| 北极星 | 内容消费量 | 内容被消费的总次数 |
| 一级 | PV、UV、互动率、分享率 | 核心内容指标 |
| 二级 | 内容分类数据、创作者数据 | 细分维度 |

## 四、图表选择指南

### 4.1 图表选择决策树

```
要展示什么？
│
├── 比较
│   ├── 分类比较 → 柱状图/条形图
│   ├── 时间趋势 → 折线图/面积图
│   ├── 部分与整体 → 饼图/环形图/堆叠柱状图
│   └── 多维度对比 → 雷达图/热力图
│
├── 分布
│   ├── 数值分布 → 直方图/箱线图
│   ├── 地理分布 → 地图
│   └── 关系分布 → 散点图/气泡图
│
├── 构成
│   ├── 静态构成 → 饼图/环形图
│   ├── 动态构成 → 堆叠面积图
│   └── 层级构成 → 树图/旭日图
│
└── 关系
    ├── 相关性 → 散点图
    ├── 流向 → 桑基图
    └── 路径 → 漏斗图
```

### 4.2 常用图表详解

#### 柱状图/条形图
| 适用场景 | 数据类型 | 注意事项 |
|----------|----------|----------|
| 分类数据比较 | 离散型 | 柱体间距一致，从0基线开始 |
| 排名展示 | 有序型 | 条形图更适合类别名称较长的情况 |
| 多系列对比 | 多维度 | 避免超过5个系列 |

#### 折线图
| 适用场景 | 数据类型 | 注意事项 |
|----------|----------|----------|
| 时间序列趋势 | 连续型 | 数据点不少于5个 |
| 多指标趋势对比 | 多维度 | 线条不超过4条 |
| 预测展示 | 预测型 | 区分实际值与预测值 |

#### 饼图/环形图
| 适用场景 | 数据类型 | 注意事项 |
|----------|----------|----------|
| 占比构成 | 分类占比 | 分类不超过6个 |
| 强调某部分 | 突出显示 | 可使用环形图突出中心数值 |
| 目标达成 | 进度展示 | 使用仪表盘更直观 |

#### 漏斗图
| 适用场景 | 数据类型 | 注意事项 |
|----------|----------|----------|
| 转化流程分析 | 流程数据 | 阶段不超过7个 |
| 识别流失环节 | 流失分析 | 标注各环节转化率 |
| 对比分析 | 多维度 | 支持多漏斗叠加对比 |

### 4.3 图表设计规范

#### 颜色使用
| 用途 | 推荐做法 | 示例 |
|------|----------|------|
| 数据系列 | 使用品牌色系，保持一致性 | 主色+辅助色 |
| 对比强调 | 使用对比色突出重点 | 红色表示下降/异常 |
| 分类区分 | 使用色相差异明显的颜色 | 避免相近色 |
| 数值映射 | 使用同色系深浅变化 | 深色=高值，浅色=低值 |

#### 坐标轴设计
| 元素 | 规范 |
|------|------|
| 刻度线 | 主要刻度线显示，次要刻度线可隐藏 |
| 刻度标签 | 避免重叠，倾斜角度不超过45° |
| 数值格式 | 大数值使用K/M/B单位，保留1-2位小数 |
| 基线 | 柱状图从0开始，折线图可截断 |

#### 图例设计
| 场景 | 推荐做法 |
|------|----------|
| 系列较少 | 图例放在图表上方或右侧 |
| 系列较多 | 使用可交互图例，支持筛选 |
| 空间有限 | 直接在数据上标注 |

## 五、布局设计

### 5.1 布局原则

| 原则 | 说明 | 实践 |
|------|------|------|
| **F型阅读** | 符合用户阅读习惯 | 重要信息放左上 |
| **网格系统** | 整齐有序 | 使用12列网格 |
| **视觉平衡** | 避免头重脚轻 | 大图表与小卡片搭配 |
| **留白呼吸** | 避免拥挤 | 组件间距一致 |

### 5.2 常见布局模式

#### 顶部指标卡+下方图表
```
┌─────────────────────────────────────────────┐
│  [指标卡1]  [指标卡2]  [指标卡3]  [指标卡4]  │
├─────────────────────────────────────────────┤
│                                             │
│              [主图表区域]                    │
│                                             │
├───────────────────┬─────────────────────────┤
│   [次图表1]       │      [次图表2]          │
└───────────────────┴─────────────────────────┘
```

#### 左右分栏布局
```
┌─────────────────────┬───────────────────────┐
│                     │                       │
│   [主图表区域]      │    [指标卡列表]       │
│                     │    [次图表1]          │
│                     │    [次图表2]          │
│                     │                       │
└─────────────────────┴───────────────────────┘
```

#### 大屏展示布局
```
┌─────────────────────────────────────────────┐
│                  [标题区域]                  │
├───────────────┬─────────────┬───────────────┤
│   [左区图表]  │  [中心图表] │  [右区图表]   │
│               │             │               │
│   [左区图表]  │  [中心图表] │  [右区图表]   │
└───────────────┴─────────────┴───────────────┘
```

### 5.3 组件尺寸建议

| 组件类型 | 最小宽度 | 推荐宽度 | 高度建议 |
|----------|----------|----------|----------|
| 指标卡 | 150px | 200-300px | 80-120px |
| 小图表 | 300px | 400-500px | 250-300px |
| 中图表 | 500px | 600-800px | 300-400px |
| 大图表 | 800px | 全宽 | 400-500px |

## 六、交互设计

### 6.1 交互层级

```
查看 → 筛选 → 下钻 → 导出
```

### 6.2 常用交互方式

| 交互类型 | 适用场景 | 实现方式 |
|----------|----------|----------|
| **悬停提示** | 查看详细数值 | Tooltip显示完整数据 |
| **图例筛选** | 显示/隐藏系列 | 点击图例切换 |
| **时间筛选** | 切换时间范围 | 日期选择器、快捷按钮 |
| **维度下钻** | 深入分析 | 点击图表元素跳转 |
| **数据导出** | 离线分析 | 导出Excel/图片 |
| **刷新设置** | 数据更新 | 手动刷新、自动刷新 |

### 6.3 筛选器设计

#### 筛选器类型
| 类型 | 适用场景 | 示例 |
|------|----------|------|
| 下拉单选 | 选项较少 | 平台、版本 |
| 下拉多选 | 多选需求 | 渠道、地区 |
| 日期选择 | 时间范围 | 近7天、自定义 |
| 数值滑块 | 数值区间 | 价格区间、年龄区间 |
| 标签筛选 | 快速切换 | 今日、昨日、本周 |

#### 筛选器布局
| 位置 | 适用场景 |
|------|----------|
| 顶部全局 | 影响整个看板的筛选 |
| 图表内部 | 仅影响当前图表 |
| 侧边栏 | 筛选项较多时 |

### 6.4 下钻设计

| 下钻类型 | 说明 | 示例 |
|----------|------|------|
| 时间下钻 | 年→季→月→日 | 点击某月查看每日数据 |
| 维度下钻 | 大类→小类 | 点击某品类查看子品类 |
| 指标下钻 | 汇总→明细 | 点击GMV查看订单明细 |
| 路径下钻 | 按流程深入 | 点击漏斗某环节查看详情 |

## 八、性能优化

### 8.1 数据加载策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 分块加载 | 先加载核心数据 | 数据量大时 |
| 懒加载 | 滚动到可视区域再加载 | 长页面看板 |
| 缓存策略 | 缓存已加载数据 | 频繁切换时 |
| 增量更新 | 只更新变化的数据 | 实时看板 |

### 8.2 图表渲染优化

| 优化项 | 方法 |
|--------|------|
| 数据量控制 | 超过1000点时采样或聚合 |
| 动画控制 | 关闭不必要的动画效果 |
| 虚拟滚动 | 大数据列表使用虚拟滚动 |
| 防抖节流 | 筛选操作添加防抖 |

### 8.3 刷新策略

| 场景 | 刷新频率 | 方式 |
|------|----------|------|
| 实时监控 | 5-30秒 | 自动刷新 |
| 日常运营 | 5-15分钟 | 自动刷新 |
| 管理决策 | 每日 | 手动刷新 |
| 历史分析 | 按需 | 手动刷新 |

## 十一、常见问题与解决方案

### 11.1 数据展示问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 数据太密集 | 数据点过多 | 采样、聚合、分页 |
| 趋势不明显 | Y轴范围不当 | 调整Y轴范围或使用对数轴 |
| 对比困难 | 图表类型不当 | 改用更适合的图表类型 |
| 占比看不清 | 饼图分类太多 | 合并小分类或改用柱状图 |

### 11.2 交互体验问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 筛选响应慢 | 数据量大 | 添加loading状态、优化查询 |
| 下钻路径不清 | 设计不明确 | 添加视觉提示、引导说明 |
| 误操作多 | 交互设计不当 | 添加确认、支持撤销 |
| 信息过载 | 内容太多 | 分页、折叠、Tab切换 |

### 11.3 性能问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 加载慢 | 数据量大 | 分块加载、缓存优化 |
| 渲染卡顿 | 图表复杂 | 简化图表、关闭动画 |
| 内存占用高 | 数据未释放 | 及时销毁、虚拟滚动 |
