## 项目描述

这是一个 AI 企业转型问卷的 H5 网页项目，旨在通过一系列调查问句帮助想要转型 AI 的企业老板进行全面的分析评估。

## 技术栈

这是一个 NextJS 全栈项目

技术栈：NextJS + TypeScript + TailwindCSS + Shadcn + React Hook Form + Zod + PostgreSQL + Prisma

部署：本地开发用 Docker，正式部署待定
