# 部署文档索引

## 说明

本目录用于沉淀两类部署知识：

1. 面向当前项目的实操文档
2. 面向后续项目可直接复用的部署模板与经验清单

目标不是只记录“一次怎么做”，而是把部署决策、执行路径、验收标准和常见坑沉淀成稳定资产。

---

## 当前文档

### 1. 当前项目专用

1. `GitHub拉取式部署指南.md`
   - 面向当前项目的部署流程优化

   - 已结合当前仓库的 `Next.js + Prisma + PostgreSQL` 约束
   - 已补充一份真实落地记录，包括本次腾讯云实例的实际软件版本、Nginx 配置、PM2 自启排障和验收结果

## 后续维护原则

后续若继续补部署文档，建议按以下规则维护：

1. 项目专用文档记录“当前仓库真实配置和命令”。
2. 通用模板只保留高复用骨架，不绑定某个具体项目。
3. 经验清单只记录稳定成立的判断，不记录一次性偶然细节。
4. 若部署方案发生升级，应同步更新三层文档，避免文档之间逐步漂移。
