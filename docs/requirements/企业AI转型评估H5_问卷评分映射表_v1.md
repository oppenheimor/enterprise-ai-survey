## 4. 得分计算

- 转型潜力原始分 = Q1 修正 + Q2 + Q3 + Q4 + Q5 + Q6
- 痛点迫切度原始分 = Q6 + Q7 + Q8 + Q13
- 落地可行性原始分 = Q2 + Q9 + Q10 + Q11 + Q12 + Q14 + Q15

换算为：

- 转型潜力得分 = 原始分归一化后映射到 40
- 痛点迫切度得分 = 原始分归一化后映射到 30
- 落地可行性得分 = 原始分归一化后映射到 30

总分为三者相加，四舍五入取整数。
